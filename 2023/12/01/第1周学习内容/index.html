星期一

clash for windows安装与使用

Clash for Windows 是目前在 Windows 上非常优秀的图形化 Clash 分支。通过 Clash API 来配置和控制 Clash 核心程序，大大降低了普通用户的使用门槛，通过它可以简单地实现科学上网。

1、下载完成后无脑下一步即可，安装完成后双击打开，双击启动会看到如下界面。

2、通过订阅链接下载配置文件，在(Profiles)配置文件一栏里面，复制购买订阅的链接 URL（这里选择一元机场），然后点击 download（下载）按钮下载订阅文件。

3、切换到 proxies（代理）一栏，会看到很多节点，选择自动选择，用鼠标点下即可，在后面可以看到延迟ms。

4、点击General，打开System Proxy项即可。

5、进入浏览器，访问Google官网，即可查看代理是否设置成功，可以看到，我的代理是新加坡。

一、JDK环境&Python环境搭建

JDK安装

1、进入JDK下载官网，找到目前最新的JDK版本，选择和操作系统一致版本的JDK，这里我选择Windows x64 installer。

Python安装

1、进入Python官网，点击Downloads选择Windows，选择需要的python版本。

2、选择和操作系统一致版本的python安装包，这里我选择Windows installer(64-bit)。

3、下载完成后，找到安装包，双击打开点击运。

4、勾选左下角Add python 版本号 to PATH，然后选择第二个自定义安装（Customize installation）。

5、选择自定义安装路径并记住，其他默认即可，点击next，等待安装完成。

6、按住快捷键win+R进入运行框，输入cmd回车，输入python回车，若显示如下则表示安装成功。

二、永恒之黑漏洞的蓝屏攻击

1、打开VMware中的win10和kali虚拟机。确保两台虚拟机正常运行。
2、在win10虚拟机上关闭防火墙，以确保后续漏洞检测和攻击操作不受阻。

3、在win10虚拟机上，通过快捷键Win+R打开命令行。
4、在命令行窗口中输入“ipconfig”命令，查看虚拟机的IPv4地址。记录下IPv4地址，后续将使用该地址进行攻击。
5、在命令行窗口中输入CVE-2020-0796-Scanner.exe路径来检测设备是否存在CVE-2020-0796漏洞。确认系统存在此漏洞，可以进行下一步操作。

6、在kali虚拟机上，使用pip install smbclient命令安装python的SMB客户端库，用于与SMB服务器进行通信。等待安装完成。
7、下载并解压CVE-2020-0796-PoC-master.zip，将其移至kali虚拟机中。该工具包含了针对CVE-2020-0796漏洞的攻击代码。

8、在kali虚拟机中打开CVE-2020-0796-PoC-master文件夹。
9、右键点击“以root身份打开”，再右键点击“在这里打开终端”。
