clash for windows安装与使用

Clash for Windows 是目前在 Windows 上非常优秀的图形化 Clash 分支。通过 Clash API 来配置和控制 Clash 核心程序，大大降低了普通用户的使用门槛，通过它可以简单地实现科学上网。

1、下载完成后无脑下一步即可，安装完成后双击打开，双击启动会看到如下界面。

2、通过订阅链接下载配置文件，在(Profiles)配置文件一栏里面，复制购买订阅的链接 URL（这里选择一元机场），然后点击 download（下载）按钮下载订阅文件。

3、切换到 proxies（代理）一栏，会看到很多节点，选择自动选择，用鼠标点下即可，在后面可以看到延迟ms。

4、点击General，打开System Proxy项即可。

5、进入浏览器，访问Google官网，即可查看代理是否设置成功，可以看到，我的代理是新加坡。
